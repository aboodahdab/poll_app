## Poll app 
### A very democratic one 
### note:right now only the backend is working
## How to run:
### first go to the backend using `cd backend`
### then make a virtual enviroment using `python3 -m venv venv`
### after that activate the virtual enviroment using `source venv/bin/activate`
### next download required libraries using `pip install -r requirements.txt`
### then run the main.py file with `python3 main.py`
### then use postman or insomnia to backend the backend
### and if frontend dropped open your browser to `localhost:3000`
